# reverse

Reverse the page order of a PDF.

## Synopsis

```
prism-docs reverse <input> [options]
```

## Options

```
-o, --output PATH      Output file path
```

## Examples

```shell
# Reverse page order
prism-docs reverse document.pdf

# Specify output
prism-docs reverse document.pdf -o reversed.pdf
```
